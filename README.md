# Password Generator

A modern, user-friendly web application to generate secure random passwords with customizable options. Built with React, Vite, and Tailwind CSS.

## Features

✨ **Customizable Password Generation**
- Adjustable password length (6-100 characters)
- Optional number inclusion
- Optional special character inclusion
- Real-time password generation

📋 **Easy to Use**
- One-click copy to clipboard
- Clean, intuitive UI
- Instant password generation on parameter change

🚀 **Built with Modern Technologies**
- React 19 for dynamic UI
- Vite for fast development and optimized builds
- Tailwind CSS for responsive styling
- ESLint for code quality
